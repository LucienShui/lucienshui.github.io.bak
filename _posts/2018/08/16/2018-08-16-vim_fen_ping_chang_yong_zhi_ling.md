## 常用指令

```bash
:sp # 上下分屏
:res y # 设置当前分屏高度为y
:vsp # 左右分屏
:vertical res x # 设置当前分屏宽度为x
Ctrl + w + hjkl # 先按Ctrl + w，然后再按hjkl在分屏之间移动
:e # 刷新当前文件
:e filename # 在当前窗口打开名字为filename的文件
```
