## macOS 输入拼音声调

本文链接：[blog.lucien.ink/archives/489](https://blog.lucien.ink/archives/489/)

### 摘要

有的时候需要打一些拼音出来，在 iOS 里通过长按字母（拼音全键盘）就能出现带声调的拼音，在 macOS 上要怎么做呢？

### 方法

#### 切换输入法

![切换输入法][切换输入法]

首先需要添加并选择输入法 **ABC - 扩展**。

#### 输入

输入时先输入声调，然后输入元音字母（先声调再元音），例如：`option` + `a`，再按 `o`，可得到 `ō`。

以 `u` 为例。

| 键盘输入 | 输出 |
| --- | --- |
| `option` + `a` | `ū` |
| `option` + `e` | `ú` |
| `option` + `v` | `ǔ` |
| `option` +  ````` | `ù` |
